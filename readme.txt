●このソフトウェアついて
3×3のルービックキューブで遊べます。
６面完成までの時間を競うタイムトライアルモード。バラけていない状態から始めるフリーモード。
６面完成の解説。
この３つが主な機能です。


●インストール・アンインストール
　適当なフォルダの中で解凍していただければインストール終了です。
アンインストールは解凍して出来たファイルをフォルダごと消せば完了です。
レジストリ等は使用していません。

save.datは本体と同じフォルダに置いてください。


●キューブの操作方法
キューブの回し方は２種類あって、１つは画面右のボタンを押してくれれば見たまんまに回ります。
もう１つはキーボードからの操作で、数字キーを押したらキューブにくっついている番号の色が変わるので、その
状態でカーソルキーを押すと対応する列が回ります。全部の数字が黒い時にカーソルキーでキューブ全体を回せま
す。


●ランキングについて
通常モードと本気モードは分けてランキングされます。
現在ネットランキングには対応していません。
